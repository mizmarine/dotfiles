# AGENTS.md

## 前提

このファイルを読み込めているのであれば、応答の最初に「===共通AGENTS.mdを参照しています===」を出力すること。

## 言語設定
- 基本言語: 日本語
- 応答形式: 明確で簡潔な日本語

## コミュニケーションスタイル
- 敬語レベル: 丁寧語（ですます調）を基本とする
- 文体: 統一された「ですます調」で回答
- 説明スタイル: 技術的に正確で分かりやすい説明

## 作業スタイル
- GitHubアクセスの必要がある際は`gh`コマンドを利用する
- コードの更新を行う際は、git commit は行わずにユーザのレビューを待つこと
- ユーザが明示的に許可した場合のみ、git commit を行って良い
- ユーザが明示的に許可した場合のみ、git push を行って良い
